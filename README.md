# Telegram Anti-Phishing Guard

Короткий опис
- Розширення для браузера (Manifest V3), яке виявляє можливі фішингові сторінки, що імітують інтерфейс Telegram, і показує попереджувальне вікно.

Швидкий старт
1. Відкрийте Chrome/Edge → Розширення → Завантажити розпаковане розширення (Load unpacked).
2. Оберіть теку з файлами розширення (папка з цим README).
3. Клікніть на іконку розширення і налаштуйте мову/увімкнення у спливаючому вікні.

Що робить розширення
- Інжектить `content.js` на всі сторінки та аналізує вміст і URL для пошуку ознак фішингу (ключові слова, поля для вводу, підозрілі домени/опечатки).
- Якщо виявлено потенційний фішинг — показує повноекранне попередження з кнопками «Покинути сайт» або «Продовжити (небезпечнo)».
- Налаштування (увімкнути/вимкнути, мова) зберігаються в `chrome.storage.sync` і керуються через `popup.html`/`popup.js`.

Файли в репозиторії
- `manifest.json` — опис розширення (manifest_version: 3). Права: `activeTab`, `scripting`, `storage`, `host_permissions: <all_urls>`.
- `background.js` — service worker, ініціалізує початкові налаштування (enabled = true, language = 'uk').
- `content.js` — основна логіка виявлення фішингу та UI попередження.
- `popup.html` — інтерфейс налаштувань (увімкнення, вибір мови).
- `popup.js` — логіка взаємодії з `chrome.storage` для збереження/завантаження налаштувань.
- `icon.png` — іконка розширення.

Технічні деталі
- Підхід виявлення: поєднання аналізу hostname (типові опечатки та підозрілі патерни), пошуку ключових слів у тексті сторінки та наявності полів для вводу (телефон/код/пароль).
- Офіційні домени Telegram перелічені у `content.js` (наприклад `telegram.org`, `t.me`, `web.telegram.org`), вони виключають показ попередження.
- Manifest V3: бекграунд реалізований як `service_worker` у `background.js`.

Обмеження і застереження
- Можливі false-positive — деякі сторінки, які згадують Telegram, можуть бути помилково помічені.
- Розширення не замінює офіційну безпеку — не надавайте свої облікові дані невідомим сайтам.

Тестування
- Завантажте розпаковане розширення у Chrome/Edge.
- Відкрийте сторінку з підозрілим доменом або створіть локальну HTML-сторінку з інпутами та текстом "Telegram" для перевірки поведінки попередження.

Пакування і публікація
- Для публікації у Chrome Web Store зпакуйте файли у ZIP або використайте панель розробника Chrome для завантаження розпакованого проекту.

Можливі покращення
- Додавати білосписок доменів користувача.
- Покращити алгоритми детекції (машинне навчання, хеші відомих фішингових URL).
- Логування подій та опціональна аналітика (з обережністю щодо приватності).
